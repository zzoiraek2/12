# Investor Plan Page

투자자에게 공유할 원페이지 시스템 기획서다.

## Local

저장소 루트에서 정적 서버를 띄운 뒤 아래 경로로 접근한다.

```bash
python -m http.server 8010
```

```text
http://localhost:8010/investor-plan/
```
